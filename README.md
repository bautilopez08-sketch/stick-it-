# Stick-it

Aplicación web para explorar, personalizar y comprar stickers.

## 🚀 Descripción

Stick-it es un proyecto enfocado en ofrecer stickers personalizados de forma simple.
El objetivo es que el usuario pueda encontrar diseños (fútbol, cultura pop, etc.) y también crear los suyos propios.

## 🛠️ Tecnologías

* Next.js
* React
* TypeScript

## ⚙️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/bautilopez08-sketch/stick-it-.git
```

Entrar a la carpeta:

```bash
cd stick-it-
```

Instalar dependencias:

```bash
npm install
```

## ▶️ Ejecución

```bash
npm run dev
```

Abrir en el navegador:

```text
http://localhost:3000
```

## 📁 Estructura

* `/app` → páginas principales
* `/components` → componentes reutilizables
* `/data` → datos del proyecto
* `/public` → imágenes y assets
